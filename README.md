# node-express-redis-demo
Simple node express application that stores login sessions in Redis

## Run in Local Dev Environement

 > install dependencies:
     $ cd sample-app && npm install

  > run the app:
     $ DEBUG=sample-app:* npm start

  > access the app
  > http://localhost:3000
